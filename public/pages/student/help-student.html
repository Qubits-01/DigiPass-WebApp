Create Gate Pass

  1. An interactive form is readily available in the dashboard/home page once you successfully sign in or create an account. It already contains fields that you can select or fill in.
  2. Fill in the information needed from the fields completely (destination, companion, departure, expected arrival, and a screenshot of parental consent). Compared to the traditional pen and paper gate passes, there is no need in filling out redundant details such as name, year level, and section as DigiPass automatically cross-references these needed data from its own database.
  3. Do not leave any blank fields; state N/A if it is not applicable to you.
  4. Once finished, you can now submit the application by clicking the button Request Gate Pass at the bottom. This will automatically send your request to your respective dorm manager for checking and approval. An automated message will also be sent to your parents/guardian upon submission.

Use Gate Pass

  1. Below the application form is the digitalized gate pass with the information you placed in the interactive form previously, along with the necessary data retrieved from the database. A status indicator is located at the upper right corner of the container.
  2. For incomplete or submitted gate pass application with issues, the status will be Rejected. On the other hand, the status will be Approved for complete and accurate submissions. If the dorm manager failed to make an action before your time of departure, the status of the gate pass application will then be Expired.
  3. When the status is finally Approved, security tags and a random QR code are both generated by the system to be later used in validation and confirmation by the guards. At the school gate, simply show the QR code to be scanned by them.
